# 深挖经历：百度笔试系统

- 简单介绍：解决了什么问题？面向哪些人？分哪几步？每一步使用了哪些技术栈？

- 最终怎么选择题目组成试卷？考虑了哪些因素？

- 最终效果好吗？怎么去检验最终的效果？
